# 🧠 Reinforcement Learning Lab

This repository is a structured hands-on learning journey into Reinforcement Learning (RL), starting from basic concepts and progressing toward Deep Reinforcement Learning.

It follows:
- Sutton & Barto: *Reinforcement Learning: An Introduction*
- Maxim Lapan: *Deep Reinforcement Learning Hands-On*

---

## 🎯 Goals

- Understand RL fundamentals deeply (not just surface-level coding)
- Implement core algorithms from scratch
- Build intuition through experiments
- Progress from tabular RL → Deep RL

---

## 📚 Learning Path

### 1. Fundamentals
- Multi-armed Bandits
- Markov Decision Processes (MDP)
- Bellman Equations
- Dynamic Programming

### 2. Classical RL
- Monte Carlo Methods
- Temporal Difference Learning
- SARSA
- Q-Learning

### 3. Deep RL
- Deep Q-Network (DQN)
- Experience Replay
- Policy Gradients
- Actor-Critic Methods

---

## 🗂 Repository Structure

- `notebooks/` → Step-by-step learning notebooks
- `src/` → Clean modular implementation of RL algorithms
- `experiments/` → Training scripts
- `configs/` → Hyperparameters and experiment settings
- `results/` → Saved models, logs, plots
